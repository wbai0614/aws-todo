# Serverless To-Do App

A fully **serverless to-do application** built on AWS using **Lambda, API Gateway, DynamoDB, and S3**.  
Easily add, complete, delete, and reorder tasks â€” all without managing any servers.

**Live Demo:** [View App](http://todo-app-12345.s3-website-us-east-1.amazonaws.com/)

---

## Features

- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Drag & drop to reorder tasks  
- Filter tasks: All / Active / Completed  
- Fully serverless and responsive UI  
- Modern design with hover effects and color-coded tasks  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript (served from S3)  
- **Backend:** AWS Lambda (Python)  
- **API Gateway:** REST API endpoints  
- **Database:** DynamoDB  
- **Hosting:** AWS S3 (Static Website Hosting)
